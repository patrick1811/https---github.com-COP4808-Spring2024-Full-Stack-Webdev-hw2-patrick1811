# Web Development Project 2 - *FlashCards*

Submitted by: **Patrick S Baker**

This web app: **The Flashcard Plant Quiz App is an interactive learning tool for plant lovers. Users can test their botanical knowledge through a series of flashcards, each featuring a plant-related question and answer. The app offers a dynamic and engaging experience, with questions ranging from basic plant facts to more advanced topics. Users can flip each card to reveal the answer and navigate through the quiz using the "Next" button. The app is visually appealing, with colorful cards and background styles based on the difficulty level of each question.
